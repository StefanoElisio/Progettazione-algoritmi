# Somma prefissa

3 approcci diversi al seguente problema:
data una sequenza di interi A = (a1, a2, ...., an).
restituiere il numero di sub-array contigui B di A.
tale che la somma degli elementi di B è pari.

= Complessità temporale =

brute_force = 0(n^3).
prefix_sum_1 = O(n^2).
prefix_sum_2 = O(n).
